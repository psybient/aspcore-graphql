## Intro 
This Project is initiated with :
`dotnet new webapp -o aspnetcoreapp`

## Requirments

`dotnet add package GraphQL --version 2.4.0`
`dotnet add package graphiql --version 1.2.0`

## run

`dotnet run`

### Use GraphiQL

Navigate to :
`https://localhost:5001/graphql/`
